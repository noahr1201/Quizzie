Fruit Ninja


Fruit Ninja is een game ontwikkeld door Halfbrick. De game is uitgebracht 21 April 2010. In de game moet je fruit dat de lucht ingegooid wordt snijden. Maar pas op want je mag niet de bommen snijden.
